Section 01
01수열
학습목표: 수열의 뜻을 설명할 수 있다.

핵심 개념 — 수열

• 차례로 나열된 수의 열을 수열이라 하고, 나열된 각 수를 그 수열의 항이라 한다.
• 각 항에 번호를 붙여 a₁, a₂, a₃, … 와 같이 나타내고, 제n항 aₙ을 일반항이라 한다.
• 수열을 일반항 aₙ을 이용하여 {aₙ}으로 간단히 나타낸다.
aₙ = (제n항을 나타내는 식) n에 1, 2, 3, … 대입 → 각 항
보기: 수열 {2n−1}의 일반항은 aₙ=2n−1이므로
a₁=1, a₂=3, a₃=5, a₄=7, … 따라서 1, 3, 5, 7, …이다.
문제 1다음 수열의 첫째항부터 넷째항까지를 나열하시오.
문제
(1) {3n+1}
(2) {2ⁿ−1}
(3) {(n−2)/(n+3)}
(1)n=1: 4, n=2: 7, n=3: 10, n=4: 13
(2)n=1: 1, n=2: 3, n=3: 7, n=4: 15
(3)n=1: -1/4, n=2: 0, n=3: 1/6, n=4: 2/7
(1) 4, 7, 10, 13    (2) 1, 3, 7, 15    (3) -1/4, 0, 1/6, 2/7
예제 1다음 수열의 일반항 aₙ을 추측하시오.
(1) 1, 1/4, 1/9, 1/16, …    (2) 9, 99, 999, 9999, …
(1) 분모 관찰a₁=1/1², a₂=1/2², a₃=1/3², a₄=1/4², …
일반항aₙ = 1/n²
(2) 각 항 분석a₁=10¹−1, a₂=10²−1, a₃=10³−1, …
일반항aₙ = 10ⁿ − 1
(1) aₙ = 1/n²    (2) aₙ = 10ⁿ − 1
문제 2다음 수열의 일반항 aₙ을 추측하시오.
문제
(1) 1×2, 2×3, 3×4, 4×5, …
(2) −1, 1, −1, 1, …
패턴을 찾아 n으로 표현. (2)는 n이 홀수/짝수에 따라 부호가 달라진다.
(1)n번째 항: n×(n+1) = n(n+1)
→ aₙ = n(n+1)
(2)홀수 항: −1, 짝수 항: 1
→ aₙ = (−1)ⁿ
(1) aₙ = n(n+1)    (2) aₙ = (−1)ⁿ
Section 02
02등차수열
학습목표: 등차수열의 뜻을 알고, 일반항과 첫째항부터 제n항까지의 합을 구할 수 있다.

등차수열의 뜻

첫째항부터 차례로 일정한 수를 더하여 만든 수열을 등차수열이라 하고, 그 일정한 수를 공차라 한다.
aₙ₊₁ = aₙ + d  (n=1,2,3,…)

등차수열의 일반항

첫째항이 a이고 공차가 d인 등차수열의 일반항 aₙ은
aₙ = a + (n−1)d
보기: ① 등차수열 2, 5, 8, 11, 14, …는 첫째항이 2이고 공차가 3이다.
② 첫째항이 5이고 공차가 −2인 등차수열은 5, 3, 1, −1, −3, …이다.
문제 1다음 등차수열의 공차와 제5항을 구하시오.
문제
(1) −1, 4, 9, 14, …
(2) 3/2, 1, 1/2, 0, …
공차 d = (이웃하는 두 항의 차). 제5항: a + 4d
(1) 공차d = 4 − (−1) = 5
(1) 제5항a₅ = −1 + 4×5 = −1 + 20 = 19
(2) 공차d = 1 − 3/2 = −1/2
(2) 제5항a₅ = 3/2 + 4×(−1/2) = 3/2 − 2 = −1/2
(1) 공차: 5, 제5항: 19    (2) 공차: −1/2, 제5항: −1/2
문제 2다음 등차수열의 일반항 aₙ을 구하시오.
문제
(1) 첫째항이 −5, 공차가 3
(2) 11, 5, −1, −7, …
aₙ = a + (n−1)d 공식 이용
(1)a = −5, d = 3
aₙ = −5 + (n−1)×3 = 3n − 8
(2) 공차d = 5 − 11 = −6, a = 11
aₙ = 11 + (n−1)×(−6) = 17 − 6n
(1) aₙ = 3n − 8    (2) aₙ = 17 − 6n
문제 3첫째항이 50이고 공차가 −4인 등차수열 {aₙ}에 대하여 다음에 답하시오.
문제
(1) 제13항을 구하시오.
(2) −10은 제몇 항인지 구하시오.
일반항aₙ = 50 + (n−1)×(−4) = 54 − 4n
(1)a₁₃ = 54 − 4×13 = 54 − 52 = 2
(2) 방정식aₙ = −10으로 놓으면
54 − 4n = −10 → 4n = 64 → n = 16
(1) 제13항: 2    (2) 제16항
예제 1제3항이 11이고 제10항이 25인 등차수열의 일반항 aₙ을 구하시오.
첫째항 a, 공차 d 설정a₃ = a + 2d = 11,  a₁₀ = a + 9d = 25
연립방정식 풀기두 식을 빼면: 7d = 14 → d = 2, a = 7
일반항aₙ = 7 + (n−1)×2 = 2n + 5
aₙ = 2n + 5
문제 4제4항이 61이고 제7항이 52인 등차수열에 대하여 다음에 답하시오.
문제
(1) 일반항 aₙ을 구하시오.
(2) 처음으로 음수가 되는 항은 제몇 항인지 구하시오.
a₄와 a₇을 이용하여 연립방정식을 세운다.
연립a + 3d = 61 ···①,  a + 6d = 52 ···②
풀기②−①: 3d = −9 → d = −3, a = 70
(1) 일반항aₙ = 70 + (n−1)×(−3) = 73 − 3n
(2) 음수 조건aₙ < 0 → 73 − 3n < 0 → n > 73/3 ≈ 24.33
따라서 처음으로 음수가 되는 항은 제25항
(1) aₙ = 73 − 3n    (2) 제25항

등차중항

세 수 a, b, c가 이 순서대로 등차수열을 이룰 때, b를 a와 c의 등차중항이라 한다.
b = (a + c) / 2  (등차중항) b − a = c − b  (공차 조건)
예제 2네 수 3, a, 15, b가 이 순서대로 등차수열을 이룰 때, a와 b의 값을 구하시오.
a는 3과 15의 등차중항a = (3 + 15)/2 = 9
15는 a와 b의 등차중항15 = (a + b)/2 → 30 = 9 + b → b = 21
a = 9, b = 21
문제 5다음 수열이 등차수열이 되도록 x와 y의 값을 정하시오.
문제
(1) −2, x, 6, y, 14, …
(2) 27, x, y, 3, −5, …
연속된 세 항 a, b, c가 등차수열 → b = (a+c)/2
(1) 공차공차 d를 구하면: 수열 −2, x, 6, y, 14에서
x는 −2와 6의 등차중항: x = (−2+6)/2 = 2
y는 6과 14의 등차중항: y = (6+14)/2 = 10
(2) 공차공차 d: 27, x, y, 3, −5에서 3과 −5 사이 공차 = −5−3 = −8 → 아니다, 전체 이용
27과 3 사이에 x, y 2개 항이 있으므로 총 3칸: 3 − 27 = 3d → d = −8
x = 27 + (−8) = 19, y = 27 + 2×(−8) = 11
확인: 27, 19, 11, 3, −5 (공차 −8) ✓
(1) x = 2, y = 10    (2) x = 19, y = 11
● 등차수열의 합

등차수열의 합 공식

등차수열의 첫째항부터 제n항까지의 합 Sₙ은
① 첫째항 a, 제n항 l일 때: Sₙ = n(a+l)/2 ② 첫째항 a, 공차 d일 때: Sₙ = n{2a+(n−1)d}/2
또, 수열 {aₙ}의 첫째항부터 제n항까지의 합 Sₙ에 대하여
a₁ = S₁,  aₙ = Sₙ − Sₙ₋₁ (n ≥ 2)
보기: ① 첫째항 5, 제10항 35인 등차수열: S₁₀ = 10(5+35)/2 = 200
② 첫째항 −1, 공차 2인 등차수열: S₁₂ = 12{2×(−1)+(12−1)×2}/2 = 120
문제 6다음 등차수열의 첫째항부터 제15항까지의 합 S₁₅를 구하시오.
문제
(1) 첫째항이 −10, 제15항이 116
(2) 첫째항이 3, 공차가 −2
(1)S₁₅ = 15×(−10+116)/2 = 15×106/2 = 15×53 = 795
(2)S₁₅ = 15×{2×3+(15−1)×(−2)}/2 = 15×(6−28)/2 = 15×(−22)/2 = −165
(1) S₁₅ = 795    (2) S₁₅ = −165
문제 7등차수열의 합 40 + 34 + 28 + ··· + (−80)을 구하시오.
문제
등차수열의 합 40 + 34 + 28 + ··· + (−80)을 구하시오.
먼저 일반항 aₙ을 구해 −80이 몇 번째 항인지 찾는다.
공차d = 34 − 40 = −6, a = 40
일반항aₙ = 40 + (n−1)×(−6) = 46 − 6n
항 수 찾기aₙ = −80 → 46 − 6n = −80 → 6n = 126 → n = 21
합 계산S₂₁ = 21×(40+(−80))/2 = 21×(−40)/2 = 21×(−20) = −420
−420
문제 8100 이상 200 이하의 자연수 중에서 5로 나누었을 때의 나머지가 2인 수의 합을 구하시오.
문제
100 이상 200 이하의 자연수 중에서 5로 나누었을 때의 나머지가 2인 수의 합을 구하시오.
조건을 만족하는 수: 102, 107, 112, …, 197 (공차 5인 등차수열)
수열 파악102, 107, 112, …, 197 (첫째항 102, 공차 5)
항 수aₙ = 102 + (n−1)×5 = 97 + 5n
aₙ = 197 → 97 + 5n = 197 → n = 20
합 계산S₂₀ = 20×(102+197)/2 = 20×299/2 = 2990
2990
문제 9첫째항부터 제5항까지의 합이 70이고 첫째항부터 제12항까지의 합이 42인 등차수열의 첫째항과 공차를 구하시오.
문제
첫째항부터 제5항까지의 합이 70이고 첫째항부터 제12항까지의 합이 42인 등차수열의 첫째항과 공차를 구하시오.
S₅S₅ = 5(2a+4d)/2 = 5(a+2d) = 70 → a + 2d = 14 ···①
S₁₂S₁₂ = 12(2a+11d)/2 = 6(2a+11d) = 42 → 2a + 11d = 7 ···②
연립②−2×①: 11d − 4d = 7 − 28 → 7d = −21 → d = −3
① 대입: a + 2×(−3) = 14 → a = 20
첫째항: 20, 공차: −3
문제 10공연장 좌석 문제
문제
900석의 객석을 갖춘 공연장을 지으려 한다. 1열의 좌석 수가 42이고, 2열부터 좌석 수는 그 앞 열의 좌석 수보다 6씩 늘어나도록 할 때, 몇 열까지 지어야 하는지 구하시오.
각 열의 좌석 수가 등차수열 (첫째항 42, 공차 6). Sₙ = 900이 되는 n을 구한다.
합 공식Sₙ = n{2×42+(n−1)×6}/2 = n(84+6n−6)/2 = n(78+6n)/2 = n(39+3n)
방정식n(39+3n) = 900 → 3n² + 39n − 900 = 0 → n² + 13n − 300 = 0
(n+25)(n−12) = 0 → n = 12 (n>0)

03등비수열
학습목표: 등비수열의 뜻을 알고, 일반항과 첫째항부터 제n항까지의 합을 구할 수 있다.

등비수열의 뜻

첫째항부터 차례로 일정한 수를 곱하여 만든 수열을 등비수열이라 하고, 그 일정한 수를 공비라 한다.
aₙ₊₁ = raₙ  (n=1,2,3,…)

등비수열의 일반항

첫째항이 a이고 공비가 r(r≠0)인 등비수열의 일반항 aₙ은
aₙ = arⁿ⁻¹
보기: ① 등비수열 4, 2, 1, 1/2, 1/4, …는 첫째항이 4이고 공비가 1/2이다.
② 첫째항이 1이고 공비가 −3인 등비수열은 1, −3, 9, −27, 81, …이다.
문제 1다음 등비수열의 공비와 제5항을 구하시오.
문제
(1) 1/25, 1/5, 1, 5, …
(2) 7, −7, 7, −7, …
공비 r = aₙ₊₁/aₙ (이웃한 두 항의 비)
(1) 공비r = (1/5)÷(1/25) = 5
(1) 제5항a₅ = (1/25)×5⁴ = 625/25 = 25
(2) 공비r = −7÷7 = −1
(2) 제5항a₅ = 7×(−1)⁴ = 7
(1) 공비: 5, 제5항: 25    (2) 공비: −1, 제5항: 7
문제 2다음 등비수열의 일반항 aₙ을 구하시오.
문제
(1) 첫째항이 −4, 공비가 3
(2) 12, −6, 3, −3/2, …
(1)aₙ = −4 × 3ⁿ⁻¹
(2) 공비r = −6÷12 = −1/2, a = 12
aₙ = 12 × (−1/2)ⁿ⁻¹
(1) aₙ = −4 × 3ⁿ⁻¹    (2) aₙ = 12 × (−1/2)ⁿ⁻¹
문제 3첫째항이 128이고 공비가 1/2인 등비수열 {aₙ}에 대하여 다음에 답하시오.
문제
(1) 제5항을 구하시오.
(2) 1/1024는 제몇 항인지 구하시오.
일반항aₙ = 128 × (1/2)ⁿ⁻¹ = 2⁷ × 2⁻⁽ⁿ⁻¹⁾ = 2⁸⁻ⁿ
(1)a₅ = 2⁸⁻⁵ = 2³ = 8
(2)2⁸⁻ⁿ = 1/1024 = 2⁻¹⁰
8 − n = −10 → n = 18
(1) 8    (2) 제18항
예제 1제2항이 −8이고 제5항이 512인 등비수열의 일반항 aₙ을 구하시오.
a, r 설정a₂ = ar = −8,  a₅ = ar⁴ = 512
나눗셈r³ = 512/(−8) = −64 → r = −4
a 구하기a×(−4) = −8 → a = 2
일반항aₙ = 2 × (−4)ⁿ⁻¹
aₙ = 2 × (−4)ⁿ⁻¹
문제 4제3항이 9이고 제6항이 243인 등비수열에 대하여 다음에 답하시오.
문제
(1) 일반항 aₙ을 구하시오.
(2) 처음으로 3000보다 커지는 항은 제몇 항인지 구하시오.
a₃, a₆ar² = 9 ···①,  ar⁵ = 243 ···②
r 구하기②÷①: r³ = 27 → r = 3
a 구하기a×9 = 9 → a = 1
(1) 일반항aₙ = 3ⁿ⁻¹
(2) 부등식3ⁿ⁻¹ > 3000
3⁷ = 2187, 3⁸ = 6561 → n−1 = 8 → n = 9
3⁸ = 6561 > 3000이므로 제9항이 처음으로 3000을 초과
(1) aₙ = 3ⁿ⁻¹    (2) 제9항

등비중항

0이 아닌 세 수 a, b, c가 이 순서대로 등비수열을 이룰 때, b를 a와 c의 등비중항이라 한다.
b² = ac  (등비중항 조건)
예제 2네 수 3, a, 12, b가 이 순서대로 등비수열을 이룰 때, 양수 a와 b의 값을 구하시오.
a는 3과 12의 등비중항a² = 3×12 = 36 → a = 6 (양수)
12는 a와 b의 등비중항12² = ab → 144 = 6b → b = 24
a = 6, b = 24
문제 5다음 수열이 등비수열이 되도록 양수 x와 y의 값을 정하시오.
문제
(1) 2, x, 10, y, 50, …
(2) 1/18, x, y, 12, 72, …
연속된 세 항 a, b, c가 등비수열 → b² = ac
(1) xx² = 2×10 = 20 → x = 2√5 (양수)

첫 번째 색칠 넓이: 1/4×4=1. 두 번째는 남은 3/4×3/4=9/16의 넓이에서 1/4을 색칠...

Sₙ₊₅ − Sₙ = aₙ₊₁ + aₙ₊₂ + ··· + aₙ₊₅. 등비수열이므로 이를 공비 r로 표현한다.

Sₙ₊₅−Sₙ = aₙ₊₁+aₙ₊₂+···+aₙ₊₅
= arⁿ + arⁿ⁺¹ + ··· + arⁿ⁺⁴
= arⁿ(1+r+r²+r³+r⁴)

∴ a(1+r+r²+r³+r⁴) = 121, r = 3
